# School Management System

A modern, high-performance School Management System built with the latest technologies.

## Features
- **Admin Dashboard**: Comprehensive management interface using Shadcn UI.
- **Student & Teacher Management**: Detailed profiles and resource tracking.
- **Real-time Updates**: Powered by Next.js Server Components.
- **Secure Data**: Integrated with MongoDB Atlas.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Components**: Shadcn UI (Admin)
- **Database**: MongoDB Atlas
- **Deployment**: Vercel

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vinayakzero/school-management.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env.local` file with the following:
   ```env
   MONGODB_URI=mongodb+srv://vk_db_user:bJDjr4OvWNMXkNkF@vk0.ksj1gsp.mongodb.net/school-management
   ```

4. **Run Development Server**:
   ```bash
   npm run dev
   ```
